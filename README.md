# Debezium CDC Course

A comprehensive production-focused course on Change Data Capture with Debezium. Learn to design and implement production-ready CDC pipelines with deep understanding of critical integration nuances.

**Live Course:** [Debezium CDC Course](https://levoel.github.io/debezium-course/)

## What You'll Learn

- Change Data Capture fundamentals and Debezium architecture
- PostgreSQL logical decoding and WAL configuration
- MySQL binlog architecture and GTID mode
- Production deployment patterns and monitoring strategies
- Stream processing with Kafka Streams and Flink
- Real-world CDC integration patterns
- GCP-native CDC architecture with Debezium Server

## Course Structure

8 comprehensive modules covering:
- Module 1: CDC Fundamentals & First Connector
- Module 2: PostgreSQL Deep Dive
- Module 3: MySQL & Aurora Integration
- Module 4: Production Operations
- Module 5: Schema Evolution & Transformations
- Module 6: Stream Processing Integration
- Module 7: GCP Deployment
- Module 8: Monitoring & Troubleshooting

## Technology Stack

- Astro 5.1
- React 19 (for interactive diagrams)
- Tailwind CSS with Apple glass design system
- MDX for course content
- Custom glass diagram primitives (zero diagram library dependencies)
