# Requirements: Debezium Course

**Defined:** 2026-01-31
**Core Value:** Инженер после прохождения курса может самостоятельно проектировать и реализовывать production-ready CDC-пайплайны на Debezium

## v1 Requirements

### Platform

- [x] **PLAT-01**: Пользователь видит интерактивный роадмап курса с визуальным отображением модулей
- [x] **PLAT-02**: Пользователь может кликать по элементам роадмапа для перехода к теме
- [x] **PLAT-03**: Прогресс пользователя сохраняется в localStorage и отображается на роадмапе
- [x] **PLAT-04**: Пользователь видит меню с разделением по модулям и топикам
- [x] **PLAT-05**: Код в контенте отображается с syntax highlighting (Python, YAML, SQL, JSON)
- [x] **PLAT-06**: Диаграммы архитектуры рендерятся из Mermaid-кода
- [ ] **PLAT-07**: Сайт деплоится как статика на GitHub Pages или Vercel *(deferred to v1.1)*
- [x] **PLAT-08**: Сайт корректно отображается на мобильных устройствах

### Module 1: Foundations

- [x] **MOD1-01**: Контент объясняет что такое CDC и log-based подход
- [x] **MOD1-02**: Контент описывает архитектуру Debezium (Kafka Connect, Server, Embedded)
- [x] **MOD1-03**: Инструкции по настройке Docker Compose окружения
- [x] **MOD1-04**: Пошаговая настройка первого PostgreSQL коннектора
- [x] **MOD1-05**: Python код для чтения CDC-событий из Kafka (confluent-kafka)
- [x] **MOD1-06**: Разбор структуры CDC-события (envelope, before/after, metadata)

- [x] **INFRA-08**: README с инструкциями по запуску

## Out of Scope

| Feature | Reason |
|---------|--------|
| Backend/API | Курс полностью статический |
| Авторизация пользователей | Прогресс только в localStorage |
| Oracle/SQL Server коннекторы | Фокус на PostgreSQL/Aurora |
| ZooKeeper deep-dive | Deprecated, Kafka использует KRaft |
| Custom connector development | Слишком продвинутая тема |
| Kafka administration | Отдельный курс, assume Kafka работает |
| Debezium UI | Experimental, не production-grade |
