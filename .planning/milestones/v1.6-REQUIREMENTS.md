# Requirements: Debezium Course

**Defined:** 2026-02-03
**Core Value:** Инженер после прохождения курса может самостоятельно проектировать и реализовывать production-ready CDC-пайплайны на Debezium с пониманием всех критических нюансов интеграций

## v1.6 Requirements

Requirements for v1.6 Full-Text Search milestone.

### Search Modal

- [x] **SRCH-01**: Cmd+K / Ctrl+K opens search modal from any page ✓
- [x] **SRCH-02**: ESC closes search modal ✓
- [x] **SRCH-03**: Input auto-focuses when modal opens ✓
- [x] **SRCH-04**: Blur overlay behind modal (glass design) ✓
- [x] **SRCH-05**: Keyboard navigation through results (↑↓ arrows) ✓
- [x] **SRCH-06**: Enter navigates to selected result ✓
- [ ] **SRCH-07**: "/" fallback trigger (for Firefox compatibility) — SKIPPED (Cmd+K + button covers all users)

### Search Behavior

- [x] **SRCH-08**: Search-as-you-type with 200ms debounce ✓
- [x] **SRCH-09**: Query highlighting in result snippets ✓
- [x] **SRCH-10**: Fuzzy search (typo tolerance) ✓
- [x] **SRCH-11**: Russian morphology support (коннектор = коннекторы) ✓
- [x] **SRCH-12**: Empty state message when no results ✓

### Search Results

- [x] **SRCH-13**: Title + contextual snippet preview ✓
- [ ] **SRCH-14**: Hierarchical grouping by module — SKIPPED (flat list cleaner UX)
- [x] **SRCH-15**: Loading indicator during search ✓

### Search Indexing

- [x] **SRCH-16**: Index all MDX lesson content at build time ✓
- [x] **SRCH-17**: Index code blocks (Python, YAML, SQL, JSON) ✓
- [x] **SRCH-18**: Index diagram tooltip content — PARTIAL (static content indexed)

### Design Integration

- [x] **SRCH-19**: Glass styling matching existing design system ✓
- [x] **SRCH-20**: Mobile responsive (full-screen on mobile) ✓

### Documentation

- [x] **SRCH-21**: Update Module 0 with search usage instructions (Cmd+K, tips) ✓

## Out of Scope

Explicitly excluded from this milestone.

| Feature | Reason |
|---------|--------|
| Server-side search | Course is fully static, no backend |
| Algolia/external search | SaaS dependency, cost |
| Search analytics dashboard | Complexity, defer to v1.7 |
| Multi-language support | Course is Russian-only |
