# Milestone v1.0: Complete Course

**Status:** ✅ SHIPPED 2026-02-01
**Phases:** 1-11
**Total Plans:** 32

## Overview

This roadmap delivers an interactive Debezium course for middle+ data engineers through 11 phases. Platform phases (1-3) establish the static website foundation with navigation and progress tracking. Infrastructure phase (4) builds the Docker Compose lab environment needed for all course modules. Content phases (5-10) follow natural learning progression from CDC fundamentals through production operations, advanced patterns, data engineering integration, and cloud-native GCP deployment. Capstone phase (11) validates end-to-end mastery with a production-ready CDC pipeline project.

## Milestone Summary

**Key Decisions:**

- Astro 5 for static site generation with React 19 islands
- Tailwind CSS 4 utility-first approach
- Shiki with github-dark theme for syntax highlighting
- nanostores for cross-island state management
- Confluent Kafka 7.8.1 (Java 21 ARM64 compatible)
- Debezium 2.5.4 (downgraded from 3.x for ARM64 stability)
- Russian explanatory text, English code/config pattern
- Groovy for Filter SMT expressions
- BACKWARD compatibility mode for Schema Registry

**Issues Resolved:**

- MDX syntax errors with `<` characters parsed as JSX (replaced with "менее")
- Java 21 ARM64 crash with quay.io/debezium images (switched to Confluent)
- Progress store array serialization issues (added safe array checks)
- Duplicate hamburger navigation display (fixed component logic)

**Issues Deferred:**

- GitHub Pages deployment (PLAT-07) — course works locally, deployment optional

**Technical Debt Incurred:**

- Shiki "promql" language warnings (fallback to plaintext acceptable)
- Phase 1 plan 04 not executed (deployment can be done manually)
